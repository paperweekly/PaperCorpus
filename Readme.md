# 关于

语料是自然语言处理的核心部件，是直接影响模型效果的重要因素。每一篇paper中都会用到一个或者多个语料进行实验，其中很多都是经典的语料，但也不乏新的高质量语料被构建并开放发布出来。这个Repo的意义在于收集已经开放发布的语料，以供研究人员方便使用。

# 字段说明

Source: 构建corpus的paper

Link: corpus的链接

Description: corpus的梗概信息

Version: 版本号和时间

Tag: corpus的标签

Organization: 发布机构

Language: 语言

Target: 使用该corpus进行实验的paper
